# data-driven-revenue-analysis
Data analysis project focused on identifying key variables impacting revenue. Includes data cleaning, visualization and predictive modeling using Python to generate business insights.

## Business Problem
Understanding which factors drive revenue is critical for making better business decisions. This project analyzes key variables to identify patterns and improve revenue prediction.

## Key Insights
- Enrollments show a strong correlation with revenue
- Certain variables have higher predictive impact
- Data visualization reveals clear trends for decision-making
