# data-driven-revenue-analysis
Proyecto de análisis de datos enfocado en identificar las variables clave que impactan el revenue. Incluye limpieza de datos, visualización y modelado predictivo utilizando Python para generar insights accionables de negocio.

## Business Problem
Entender qué factores impulsan el revenue es fundamental para tomar decisiones estratégicas efectivas. Este proyecto analiza variables clave para identificar patrones y mejorar la precisión en la predicción de ingresos.

## Key Insights
-Los enrollments (inscripciones) muestran la relación más fuerte con el revenue, siendo el principal motor de crecimiento
-Las métricas de conversión tienen mayor impacto que las métricas de tráfico
-Los diferentes canales de marketing aportan valor de forma distinta a lo largo del funnel, equilibrando volumen y eficiencia
-La visualización de datos permite identificar patrones claros que respaldan la toma de decisiones basada en datos
